About


  • This website is a straightforward port-scanning service online using Nmap that analyzes the provided website for potential security flaws and displays the report.
  • This website is designed to assist non-technical website managers to easily evaluate their websites' levels of security without having to learn Bash and install complicated terminal-based scanners on a Linux operating system.
  • By serving as a simple scanner that is accessible online and effortless to use, this website strives to contribute in raising awareness of web security concerns over the Internet.
